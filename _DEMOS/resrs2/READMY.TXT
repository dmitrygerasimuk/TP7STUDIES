  1. Введение
  ───────────
    Данная программа предназначена для визуального построения 
диалоговых интерфейсных элементов с пользователем для библиотеки 
Turbo Vision и совместимых с ней.
    Возникла потребность в данной программе при написании диалоговых 
интерфейсов с пользователем.
    Автор хочет в очередной раз восхититься библиотекой TV, чтоль 
удачной и к месту созданной Borland Inc.

  2. Требования к системе
  ───────────────────────
    В данный момент это программа собирается различными компиляторами с 
ЯВУ Паскаль (BP7, VP2b243, FPC 1.4, Delphi 2-5)
    В зависимости от используемого компилятора соответсвенно именяются
требования к системе. В наиболее универсальном варианте (сборка BP70, 
real mode) для нормального функционирования необходимо 560-600 кил 
свободной оперативной памяти, программа кэширования диска типа SmartDrv,
и манипулятор мышь.

  3. Системные возможности
  ────────────────────────
    Ограничения в программе также зависят от используемой платформы, 
наименее функциональна сборка BP, real mode - по причине малого количества 
памяти, в защищённом режиме практически все органичения по памяти снимаются.
    Собранная под платформу Win32 программа становится полномасштабным Win32
приложением.

   Ограничения:                 │ BP RM   │   BP PM   │  Win32 (VP, FPC, D2-5)
   ═════════════════════════════╪═════════╪═══════════╪═══════════════════════
   Количество форм в программе  │ 16536   │   16539   │   2^32
                                │         │           │
   Объём редактируемого текста  │ ~10 мег │   2 Гб    │   2^64
    в редакторе                 │         │           │

  4. Идеи
  ───────
    Очень сильное влияние на принципы работы оказала Delphi, можно сказать
  что это был орентир для меня.
    Для собственного редактора текстов я хорошо изуил редактор, который идёт 
  в IDE FPC (хорошая вещь Open Sourse)
    Для макросов используется адаптированый вариант инетрпритатора с 
  Pascal-script языка (автор Циганенко Александр).

  5. Приступая к работе
  ─────────────────────
    Для быстрого ознакления с программой можно попытаться воспользоваться 
  следущей  инструкцией:
    5.1. Для создания формы:
      - после запуска программы 
        в левом нижнем углу экрана будет окно Редактора свойств (он аналогичен 
        своему прототипу из дельфи)
      - нажмите клавишу F7 - перед вами появится фрма дизайнера
      - нажмите F4 - перед вами выскочит диалог, сожержащий списки доступных
        визуальных и не визуальных компонет для построения диалога
      - выберите например InputLine и нажмите Enter
        объект TInputLine будет помещён на нашу форму
      - для изменения его свойств, размеров, местоподения щёлкните по нём 
        правой кнопкой мишки (сочетание клавиш Shift+F2), внизу его
        появится двойная белая горизонтальная линия - курсор дизайнера
        за него можно мышкой перемещать оъект по форме, а если взять за
        правый одинарный угол курсора - то и менять размеры объекта
      - при выборе курсором объекта в окне инспеткора свойств появться 
        доступные для изменения свойства объекта
      - для создания исходного текста формы выберите меню 
        "Проект"/"Создать флорму"
        откроется окно в котором будет содержаться тсходный текст нашего 
        диалога
